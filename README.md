# SB-Admin-CLI-Release

SB-Admin-CLI is a powerful command-line tool to manage your SelfBest account, attendance, leave, teams, and more â€” all from your terminal.

---

## Features

- **User Authentication**  
  - Sign in using Google OAuth or email/password  
  - Login, logout, and password update commands  


- **Leave Management System (LMS)**  
  - View leave balance and history  
  - Apply and cancel leave requests  
  - Access leave bureau and public holidays  

- **Attendance Management**  
  - Record and regularize attendance  
  - View attendance reports  

- **People & Organization**  
  - Search and view organizational structure  
  - Lookup employee profiles  


- **SelfBest CLI Updates**  
  - Update CLI to the latest version  
  - Uninstall CLI  


---

## Installation

### Windows

Open **PowerShell** and run:

```powershell
iwr -useb https://raw.githubusercontent.com/tiwari-mani-tft/SB-Admin-CLI-Release/main/install.ps1 | iex
```

### Linux/Mac

Open **terminal** and run:

```terminal
curl -sSL https://raw.githubusercontent.com/tiwari-mani-tft/SB-Admin-CLI-Release/main/install.sh | bash
```
